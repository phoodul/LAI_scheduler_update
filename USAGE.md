# 사용 설명서 (Usage Guide)

## 1. 기본 사용법

### 신규 처방 입력
- 날짜 칸을 **더블 클릭**하거나 **F1** 키를 눌러 신규 처방을 입력합니다.
- 환자명 입력 후 **[+ 주사 항목 추가]** 버튼을 눌러 여러 개의 약물을 동시에 처방할 수 있습니다.
- 약품명 아래에 **용량**과 **간격(일)**을 입력합니다.
- 메모를 입력 후 **[등록/수정]** 버튼을 누릅니다.
- 저장 시 주사 기록(파란 블록)과 다음 예약(주황 블록)이 자동으로 생성됩니다.

### 예약 완료 처리 (당일 방문)
- 달력에 주황색으로 표시된 **[예정]** 블록을 더블 클릭합니다.
- 내용 확인 또는 수정 후 **[등록/수정]** 버튼을 누르면 해당 날짜에 주사 기록이 생성되고, 다음 예약이 자동으로 계산됩니다.

### 처방 기록 수정
- 달력에 녹색으로 표시된 **[주사]** 블록을 더블 클릭합니다.
- 하나의 기록 내에서 약품을 추가하거나 삭제할 수 있으며, 용량 및 간격을 수정하여 저장하면 연결된 각각의 예약일도 함께 업데이트됩니다.
- 예약이 삭제된 상태에서 주사 기록을 수정하면 새로운 예약이 자동으로 재생성됩니다.

## 2. 예측 및 모니터링 기능 (Drug Adherence)

환자별 약물 순응도를 한눈에 시각적으로 파악할 수 있는 신규 기능입니다.

1. 화면 하단의 **[Drug Adherence]** 버튼을 클릭합니다.
2. 팝업창 상단에서 **환자 선택** 및 **약물 선택** 드롭다운을 통해 특정 환자와 약물을 고릅니다.
3. 조회할 **시작일**과 **종료일**을 지정하고, 기대 주사 간격인 **주기(일)**를 입력한 뒤 **[결과 보기]**를 누릅니다.
4. 아래와 같은 결과가 나타납니다:
   - **전체 기간, Total Days Late:** 선택한 기간 동안 지연된 환자의 총 주사 공백 일수.
   - **PDC (Proportion of Days Covered):** 총 기간 중 실제로 약물 효과가 지속된 일수의 비율을 퍼센트(%)로 표시 (예: 91.7%).
   - **타임라인 시각화:** 달력 전체 뷰 아래에 일자별로 투여 기록이 그려집니다. 약물별 고유 색상이 적용되며, 권장 주기보다 겹치게 맞았을 경우 (오버랩) 캔버스에 위/아래로 번갈아가며 겹침 상태를 직관적으로 나타냅니다.

## 3. 예약일 드래그로 변경

환자가 예약일보다 일찍 또는 늦게 방문한 경우, 예약 블록을 드래그하여 날짜를 수정할 수 있습니다.

1. 주황색 **[예정]** 블록 위에서 **Ctrl + 마우스 클릭**을 유지합니다.
2. 원하는 날짜 칸으로 **드래그**합니다. 이동 가능한 날짜는 파란색으로 하이라이트됩니다.
3. 마우스를 **놓으면** 예약이 해당 날짜로 이동하고, 이전 주사 기록의 간격도 실제 날짜 차이로 자동 업데이트됩니다.
4. 이동된 날짜에서 블록을 **더블 클릭**하여 처방을 완료하면, 다음 예약이 수정된 간격 기준으로 생성됩니다.

## 4. 기록 삭제 및 알림 기능

### 기록 삭제
- 날짜 칸을 더블 클릭하면 하단에 해당 날짜의 기록 목록이 표시되고, 각 항목 오른쪽의 **[삭제]** 버튼으로 삭제할 수 있습니다.
- **[주사] 기록** 삭제 시 연결된 다음 예약도 함께 삭제됩니다.
- **[예정] 예약**만 클릭해 삭제하면 이전 주사 기록은 유지됩니다.

### 팝업 알림
프로그램 시작 시 및 매일 오전 7시에 예약 환자 알림 팝업이 자동으로 뜹니다.
- **오늘(분홍), 내일(주황), 모레(파랑)**
- 하단의 알림 체크박스로 on/off가 가능합니다. 단축키 `Alt + K`로 하나씩 닫거나, `Shift + Alt + K`로 전체 닫기가 가능합니다.

## 6. 스마트폰 구글 캘린더 자동 연동 (OAuth 2.0)

이 프로그램은 사용자의 구글 캘린더와 안전하게 동기화되는 기능을 지원합니다. `Janssen LAI Scheduler`라는 전용 달력이 구글 계정 내에 생성되므로 개인 일정이나 다른 사용자의 데이터와 섞이지 않습니다.

1. 하단의 파란색 **[구글 캘린더 연동]** 버튼을 클릭합니다.
2. 인터넷 창이 열리면 원하는 구글 계정을 선택합니다.
3. **"구글에서 이 앱을 확인하지 않았습니다"** 라는 경고 화면이 느면:
   - 화면 하단의 **[고급]** 버튼을 클릭합니다.
   - **"Janssen LAI Scheduler(으)로 이동(안전하지 않음)"** 링크를 클릭합니다.
   - 이 경고는 구글의 공식 심사를 받지 않은 내부 앱에서 정상적으로 표시되는 화면으로, 정보 유출의 위험은 없습니다.
4. **[허용]**을 누르면 인증 완료! 이후 예약이 스마트폰 구글 캘린더에 자동으로 반영됩니다.

> **참고:** ICS 저장 폴더 기능은 오프라인용 달력 백업(schedule.ics) 추출용으로 여전히 사용할 수 있습니다.
